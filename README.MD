# MERN Task Servidor

## Descripcion

- API Rest hecho en NodeJS/Express utilizando base de datos MongoDB.
- En la DB esta compuesta por tres tablas: USUARIOS/PROYECTOS/TAREAS

### Dependenciias

- Clonar el proyecto y ejecutar le comando:

```
 npm install
```

### Ejecucion del programa

```
npm run dev
```
